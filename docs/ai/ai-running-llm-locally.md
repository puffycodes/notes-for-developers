# Running LLM Locally

## Docker Model Runner

Reference: [Docker Model Runner](https://docs.docker.com/ai/model-runner/)

Test:
```
docker model version
docker model run ai/smollm2
```

### Examples

Run OpenAI gpt-oss with a single command:

```
docker model run ai/gpt-oss:20B
```
