## Installation

### On Linux and MacOS (With Admin Rights)

```bash
curl -fsSL https://openclaw.ai/install.sh | bash
```

### Install Without Admin Rights

From Git:
```bash
git clone https://github.com/OpenClaw/OpenClaw.git
cd OpenClaw
npm install
node scripts/run-node.mjs onboard
```
