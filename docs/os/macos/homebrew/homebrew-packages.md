# Homebrew Packages

## Caching Github Credential

References: https://docs.github.com/en/get-started/getting-started-with-git/caching-your-github-credentials-in-git

Install Git: (not done in this way because git already come by default on MacOS)
```
brew install git
```

Install Git Credential Manager (GCM):
```
brew tap microsoft/git
brew install --cask git-credential-manager-core
```

## feroxbuster

Feroxbuster is installed for used as part of autorecon

Install feroxbuster:
```
brew install feroxbuster
```

## Tools

Install tools such as diff:
```
brew install diffutils
```
