### Simple Steps

Initialize the directory:
```
terraform init
```

Format and validate:
```
terraform fmt
terraform validate
```

Create infrastructure:
```
terraform apply
```

Show state:
```
terraform show
```

Take down the infrastructure:
```
terraform destroy
```
