# Docker Installation

## For MacOS

### Installation

1. Download from [here](https://docs.docker.com/desktop/setup/install/mac-install/)

    1. Open Docker.dmg
    1. Drag Docker.app to Application
    1. Start Docker.app and follow the instruction

1. Install Rosetta 2

    ```
    softwareupdate --install-rosetta
    ```

### Upgrade

1. Upgrade from Docker.desktop when notified
