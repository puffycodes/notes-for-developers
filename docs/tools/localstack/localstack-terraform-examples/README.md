# Terraform for LocalStack Examples

## Install tflocal

1. Install Terraform
1. [Install tflocal](../localstack-installation.md#install-tflocal)

## Simple S3 Bucket

To use:

```
cd simple-s3-bucket
tflocal init
tflocal plan
tflocal apply
tflocal state list
tflocal destroy
```
