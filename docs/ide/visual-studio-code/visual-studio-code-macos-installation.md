# Visual Studio Code Installation

## MacOS

### Installation

1. Download from [here](https://code.visualstudio.com/download)
1. Extract all from the .zip file.
1. Move Visual Studio Code.app to Application folder.

### Launching from the command line

1. Launch VS Code
2. Open Command Palette and type 'shell command' to find "Shell Command: Install 'code' command in Path" command
3. Restart the terminal

4. To remove, use "Shell Command: Uninstall 'code' command from PATH"

### Extra

1. Install Python extension (which include jupyter notebook extension)
