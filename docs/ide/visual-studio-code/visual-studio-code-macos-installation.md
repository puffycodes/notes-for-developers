## MacOS

### Installation

1. Download from [here](https://code.visualstudio.com/download)
1. Extract all from the .zip file.
1. Move Visual Studio Code.app to Application folder.

Note: New installer comes as .dmg file.

### Update

1. Vistual Studio Code will check for updates.

### Launching from the command line

Adding path:
1. Launch VS Code
1. Open Command Palette and type 'shell command' to find "Shell Command: Install 'code' command in Path" command
1. Restart the terminal

Removing path:
1. To remove, use "Shell Command: Uninstall 'code' command from PATH"

### Extra

1. Install Python extension (which include jupyter notebook extension)
