# Pnpm Installation

Preferred Method: Using Corepack (included with node.js):
```bash
corepack enable
corepack prepare pnpm@latest --activate
```

Note: ```corepack enable``` needs to run in Administrator mode on Windows.

Alternative Method: Using npm (Global Install):
```bash
npm install -g pnpm
```
