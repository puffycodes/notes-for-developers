# Python Packages for Financial

## Data Source

1. [Financial Modeling Prep](https://financialmodelingprep.com)

## Packages

### Market Data

1. [OpenBB - Access Market Data](https://pypi.org/project/openbb)
    1. Install Python Package
        ```
        pip install openbb
        ```
    1. [Documentation](https://docs.openbb.co/python)
    1. Additional Extensions
        1. OpenBB-Technical
            ```
            pip install openbb-technical
            ```

1. [FinanceToolkit](https://github.com/JerBouma/FinanceToolkit)
    1. Install Python Package
        ```
        pip install financetoolkit
        ```

### Technical Indicators

1. [TA-Lib - Technical Analysis](https://ta-lib.org)
    1. [Install TA-Lib](https://ta-lib.org/install)
        1. For MacOS
            ```
            brew install ta-lib
            ```
    1. [Install Python Package](https://github.com/TA-Lib/ta-lib-python.git)
        ```
        pip install TA-Lib
        ```

### Visualization

1. [Plotly](https://plotly.com/python/)
    ```
    pip install plotly
    ```
1. [Dash - Web-Base UI for Plotly](https://dash.plotly.com/)
    1. Installation
        ```
        pip install dash
        ```
    1. [Dash Tutorial](https://dash.plotly.com/tutorial)
